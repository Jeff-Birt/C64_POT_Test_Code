## Test code for C64 POT inputs

Adapted from the C64 Programmers Reference Guide. There are four versions:

* Read all 4 pot inputs at once continiously
* Read 100 samples from all 4 inputs, show Avgerage, Low and High readings
* Read 100 samples from AX input, show Avgerage, Low and High readings
* Read 100 samples from AY input, show Avgerage, Low and High readings

You are free to use this code. Please attribute to the source.
